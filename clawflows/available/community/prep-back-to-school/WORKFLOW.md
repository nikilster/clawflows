# Prep Back to School

Everything for the new school year, organized into manageable steps. Supplies, schedules, lunches, carpools — handled.

## 1. Gather School Details

Ask for the basics and save to `~/.openclaw/data/school/`: kids' names and grades, school start date, supply lists, teacher names, school hours, and after-school activities.

## 2. Build the Supply Checklist

Organize by kid. Check what's on hand vs. what's needed:

```
📝 Supplies — Emma (4th Grade)

□ Backpack — has one, still good ✅
□ Lunchbox — needs new
□ 24-pack crayons    □ Glue sticks (4)
□ 3 notebooks        □ 2 pocket folders
□ #2 pencils pack    □ Scissors (blunt)

ALREADY HAVE: 3 items | STILL NEED: 8 items (~$40)
```

Suggest checking what's at home before buying everything new.

## 3. Set Up the Schedule

Using your **calendar skill**, build the weekly schedule: drop-off/pick-up times, after-school activities, recurring events (early dismissals, PTA), and homework time blocks.

## 4. Plan Lunches and Carpool

Build a simple 5-day lunch rotation with a grocery list. If carpooling, organize who drives which days and add it to the calendar.

## 5. Present the Master Checklist

```
📝 Back to School — August 2026

KIDS: Emma (4th), Jake (1st)

📋 SUPPLIES: Emma ~$40, Jake ~$55 — shop by Aug 20
📅 SCHEDULE: school hours + activities added to calendar
🍎 LUNCHES: Week 1 rotation ready
📦 STILL TO DO
   □ Buy supplies    □ Label everything
   □ Confirm carpool □ Adjust bedtimes this week

⏰ 12 days until school starts
```

Send via your **messaging skill**.

## Notes

- Every family is different. Follow the user's lead on what matters.
- Check if the school has specific brand requirements for supplies.
- Label everything. Seriously — put names on every single item.
- The first week is always chaotic. Set expectations accordingly.
- Save this year's setup to make next year easier.
