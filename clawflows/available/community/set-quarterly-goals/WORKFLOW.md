# Set Quarterly Goals

A structured goal-setting session to reflect on the past quarter and set clear, achievable goals for the next three months.

## Steps

1. **Review last quarter** — Look back at the past 3 months. Use your **calendar skill** to surface major events and milestones. Ask the user what they accomplished, what they're proud of, and what fell through the cracks.

2. **Identify themes** — What areas of life matter most right now? Common buckets: career, health, relationships, finances, learning, personal projects, fun.

3. **Set 3-5 goals** — Work with the user to define goals for next quarter. Each goal should be:
   - Specific enough to know when it's done
   - Achievable in 3 months
   - Something the user actually cares about (not just "should" do)

4. **Define measurable outcomes** — For each goal, set a clear finish line. "Get healthier" becomes "Exercise 3x/week for 12 weeks." "Save money" becomes "Save $1,500 by June 30."

5. **Plan first steps** — For each goal, identify one concrete action to take this week to get started.

## Output Format

```
Quarterly Goals — [quarter and year]

Last Quarter Recap:
  ✅ Completed: Got promoted, ran first 10K
  🔄 In Progress: Learning guitar (keep going!)
  ❌ Dropped: Didn't start the blog

Goals for Next Quarter:

1. Run a half marathon
   Measure: Complete a half marathon by March 31
   First step: Register for a race this week

2. Save $2,000 for vacation fund
   Measure: $2,000 in savings account by March 31
   First step: Set up automatic $170/week transfer

3. Read 4 books
   Measure: Finish 4 books (any genre)
   First step: Pick the first book tonight

4. Launch personal website
   Measure: Live site with 3 pages
   First step: Buy a domain name this weekend
```

## Notes

- Three to five goals is the sweet spot. More than five and focus gets diluted.
- Goals should excite the user, not stress them out. If something feels like a chore, dig into why.
- Write goals down. The act of writing them makes them more real.
- Suggest pairing this with the review-month workflow for monthly check-ins on progress.
