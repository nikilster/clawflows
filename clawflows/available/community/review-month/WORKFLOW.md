# Review Your Month

A guided monthly reflection to look back on what happened, celebrate progress, and make adjustments for the month ahead.

## Steps

1. **Recap the month** — Use your **calendar skill** to review major events, meetings, and milestones from the past 30 days. Summarize what the month looked like at a high level.

2. **What went well** — Ask the user what they're proud of or happy about this month. Prompt with specific areas: work, health, relationships, personal projects, learning.

3. **What was tough** — Ask about challenges or things that didn't go as planned. Keep it constructive — focus on what can be learned, not what went wrong.

4. **Goal progress** — If the user has goals (quarterly, annual, or ongoing), check in on each one. Note progress, stalls, or completions.

5. **Adjustments for next month** — Based on the reflection, suggest 1-3 small adjustments. Maybe a habit to start, something to stop, or a priority to shift.

## Output Format

```
Monthly Review — [month]

Highlights:
  - Launched the new project at work
  - Ran 3x per week for the whole month
  - Finished that book you started in January

Challenges:
  - Sleep schedule got messy mid-month
  - Didn't make progress on the side project

Goal Check:
  Read 12 books this year — 4/12 (on track ✅)
  Run a half marathon — training consistent ✅
  Learn Spanish — paused ⏸️

Next Month:
  - Set a consistent bedtime (try 11pm)
  - Block 2 hours on weekends for side project
  - Restart Spanish — even 10 min/day counts
```

## Notes

- This is a personal reflection, not a performance review. Keep it warm and supportive.
- Don't push too hard on areas where the user is struggling. Acknowledge difficulty.
- If the user hasn't set explicit goals, help them notice patterns from the conversation.
- This pairs well with the set-quarterly-goals workflow for bigger-picture planning.
