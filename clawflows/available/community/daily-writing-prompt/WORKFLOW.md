# Daily Writing Prompt

Start your day with a spark of creativity. This workflow delivers a new writing prompt each morning — rotating between fiction, memoir, poetry, and observation exercises — and keeps track of your writing streak.

## Steps

1. Check what type of prompt was delivered last time (fiction, memoir, poetry, or observation) and rotate to the next type.
2. Generate a fresh, thoughtful prompt that's specific enough to be inspiring but open enough for personal interpretation.
3. Check the user's writing streak — how many consecutive days they've written something.
4. Present the prompt with the streak count and a bit of encouragement.
5. When the user writes their response (any length is fine), save it using your **notes skill** with the date and prompt type tagged.

## Output Format

```
✍️ Daily Writing Prompt — Thursday, March 20
   Type: Memoir | Streak: 12 days 🔥

Today's Prompt:
Write about a meal that changed something for you. Maybe it was the
first time you tried something new, a conversation that happened over
dinner, or a meal you ate alone in a place far from home. Focus on
one sensory detail and let the memory unspool from there.

(Write as much or as little as you'd like. Even a few sentences count.)
```

## Notes

- Rotate prompt types in order: fiction, memoir, poetry, observation. Then repeat.
- Any response counts toward the streak — even a single sentence. The goal is consistency, not perfection.
- Keep prompts accessible for all skill levels. Avoid jargon or literary terms.
- If the user misses a day, reset the streak gently without making them feel bad about it.
- Save all entries so the user can look back on their writing over time.
