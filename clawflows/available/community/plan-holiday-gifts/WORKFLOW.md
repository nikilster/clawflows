# Plan Holiday Gifts

Get ahead of holiday gift-giving. Build your list, set budgets, track purchases, and never scramble at the last minute.

## 1. Build the List

Check for an existing list at `~/.openclaw/data/gifts/`. If starting fresh, ask: who you're buying for, total budget, and for each person — relationship, interests, and individual budget. Save so it gets smarter each year.

## 2. Suggest Ideas

For each person, suggest 2-3 specific gifts based on their interests, age, budget, and what was given in past years. Mix price points: one at-budget, one under, one creative or homemade option.

## 3. Track Purchases

As the user buys gifts, update the tracker with what was purchased, cost, wrapping status, and shipping deadlines.

## 4. Present the Tracker

```
🎁 Gift Tracker — Updated March 20

BUDGET: $500 total | $310 spent | $190 remaining

PERSON         BUDGET   STATUS       GIFT
Mom            $75      ✅ Bought    Ceramic planter ($62)
Dad            $75      ✅ Bought    Grilling tools ($71)
Sister         $60      🔲 Ideas     Yoga mat? Gift card?
Nephew (8)     $40      🔲 Pending   LEGO set — check sales
Coworker       $25      🔲 No ideas  Need suggestions

📦 Nephew's gift needs to ship by Dec 15
💡 Sister is into yoga and mystery novels
```

Send updates via your **messaging skill** whenever the list changes or deadlines approach.

## Notes

- Never share anyone's gift list with anyone else. Privacy matters.
- If the user is over budget, gently suggest swaps or homemade alternatives. Don't lecture.
- Save this year's list to help with next year — "Last year you got Mom a planter" is gold.
- Works for any occasion, not just December holidays. Birthdays, Mother's Day, graduations — adapt freely.
- Respect that not everyone celebrates the same holidays. Follow the user's lead.
