# Spring Clean

A room-by-room deep-cleaning guide that breaks the big job into manageable daily chunks. Track progress and tackle one room at a time.

## 1. Set Up

Check for a clean in progress at `~/.openclaw/data/spring-clean/`. If starting fresh, ask how many rooms to cover and how many days to spread it over. Default rooms: kitchen, bathrooms, living room, bedrooms, closets/storage, garage/basement.

## 2. Build the Checklist

For each room, create a focused list of deep-clean tasks (not everyday cleaning). Examples:

- **Kitchen** — clean inside oven, wipe cabinets, clean under sink, organize pantry, toss expired items
- **Bathroom** — scrub grout, clean exhaust fan, wash shower curtain, check expired meds
- **Living Room** — vacuum under furniture, clean windows, dust ceiling fans, wipe electronics
- **Bedroom** — wash pillows, vacuum mattress, rotate mattress, clean under bed, sort clothes for donation

## 3. Track Progress

```
🧽 Spring Clean — Day 3 of 7

TODAY: Kitchen 🍳
□ Clean inside oven          □ Organize pantry
□ Wipe cabinet fronts        □ Clean range hood filter
□ Clean under sink           □ Wipe light fixtures

PROGRESS
✅ Living Room  ✅ Bathrooms  🔲 Kitchen (today)
🔲 Bedrooms  🔲 Closets  🔲 Garage

📊 2 of 7 rooms done — 29% complete
```

## 4. Deliver

Send daily progress and final celebration via your **messaging skill**.

## Notes

- One room per day is a suggestion. Some people do it all in a weekend, others take two weeks. Adapt.
- If the user skips a day, don't guilt-trip them. Just pick up where they left off.
- Keep checklists practical — 6-10 tasks per room, not 30.
- This isn't everyday cleaning. Focus on stuff that only gets done a few times a year.
- Seasonal timing is flexible. Spring clean in October if that's when motivation strikes.
