# Prep for an Event

Got a party, conference, wedding, or dinner to attend? This workflow helps you figure out everything you need to know beforehand so you show up feeling confident and prepared.

## Steps

1. Ask the user which event they're preparing for. Check your **calendar skill** for upcoming events if they're not sure.
2. Gather details: event type, location, time, dress code (if any), and who else is going.
3. Use your **browser skill** to research the venue, parking/transit options, and anything useful to know (menu, agenda, layout).
4. Based on the event type and dress code, suggest what to wear.
5. Build a checklist of things to bring, logistics to know, and any conversation starters or prep that would help.
6. Present everything in one clear briefing.

## Output Format

```
🎉 Event Prep: Sarah's Housewarming Party

📅 Saturday, March 22 at 6pm
📍 412 Oak St, Apt 3B — street parking available
👥 ~20 people, casual friends group

👔 What to Wear:
- Casual — jeans and a nice top/button-down
- It's a 3rd-floor walkup, wear comfortable shoes

🎁 What to Bring:
- Housewarming gift: candle, wine, or small plant ($15-30 range)
- BYOB — Sarah mentioned it's not fully stocked yet

📝 Good to Know:
- Sarah's partner's name: Taylor (they/them)
- They just moved from Chicago
- Sarah mentioned they adopted a cat named Mochi
```

## Notes

- Tailor suggestions to the formality of the event. A wedding needs different prep than a BBQ.
- If the user knows other attendees, pull in any conversation context from past notes if available.
- For professional events, include relevant names, titles, and talking points.
- Keep gift suggestions practical and within a reasonable budget unless the user specifies otherwise.
