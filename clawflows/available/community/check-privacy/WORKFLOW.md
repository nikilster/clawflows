# Check Privacy

A quarterly privacy check-up. See what has access to your data, tighten permissions, and clean up things you forgot you authorized.

## 1. Review App Permissions

Walk through key device permissions — location, contacts, camera, microphone, photos. For each, ask: "Does [app] still need access to [permission]?" Suggest changing "Always" to "While Using" for location where possible.

## 2. Check Connected Accounts

Review third-party apps connected to main accounts:

- **Google** — Settings > Security > Third-party apps
- **Apple ID** — Settings > Password & Security > Apps Using Apple ID
- **Facebook** — Settings > Apps and Websites

Flag anything unrecognized, unused, or with broad permissions.

## 3. Review Privacy Settings

Check key settings on most-used services: social media visibility, search history, ad personalization, smart home voice recording storage, browser tracking protection.

## 4. Present the Report

```
🔒 Privacy Audit — March 2026

📱 APP PERMISSIONS
   12 apps have location access (3 set to "Always")
   8 apps can read contacts (2 unnecessary)

🔗 CONNECTED ACCOUNTS
   Google: 14 apps (3 unrecognized)
   Facebook: 9 apps (4 unused 6+ months)

📋 ACTION PLAN
   1. Revoke 3 unrecognized Google connections
   2. Remove 4 unused Facebook connections
   3. Change 3 apps to "While Using" location
   ⏱️ ~15 minutes total
```

Send via your **messaging skill**.

## Notes

- **NEVER change settings without explicit approval.** Recommend, then let them decide.
- **NEVER access or log into accounts on the user's behalf.**
- Privacy is personal. Some people are fine with tracking for convenience. Don't push.
- Focus on quick wins first — revoking unused connections is easy and high-impact.
- If the user feels overwhelmed, pick the top 3 actions and save the rest for next time.
