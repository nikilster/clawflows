# Support Friend Announcement

## 1. Get the post URL

Use the X/Twitter post URL the user provides.

If no valid post URL is provided, stop and ask for one.

## 2. Open the post in the signed-in account

Open the post in the browser/account already connected on this machine.

If the account is not signed in or the post cannot be accessed, stop and report the failure clearly.

## 3. Perform support actions

Do these actions in order:

1. Like the post
2. Repost / retweet the post
3. Add a short supportive reply

Reply guidelines:
- Keep it short and natural
- Be supportive and generic
- Do not mention private details
- If the post already appears liked/reposted/replied from this account, do not duplicate the action; report it as already done

## 4. Verify the result

Confirm which actions succeeded:
- liked
- reposted / retweeted
- replied

Capture the final post URL after actions are complete.

## After Running

Send the user a short confirmation with:
- the original post URL
- which actions were completed
- the exact reply text that was posted (if any)
- any action that was skipped or failed

Do not include any private or sensitive information in the confirmation.
